## Usage

Simply add to your `pawn.json` and include:

```json
{
    "dependencies": [
        "ScavengeSurvive/LIBRARY_NAME"
    ]
}
```

```pawn
#include <LIBRARY_NAME>

#include "your-library.inc"
```

## Testing

To test, simply run the package:

```bash
sampctl package run
```

Then connect to `localhost:7777`.
